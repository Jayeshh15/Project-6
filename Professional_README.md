## 📌 Project Overview

The **Personal Journal Manager** is a Python-based application built
using **Object-Oriented Programming (OOP)** concepts. It allows users to
manage their journal entries efficiently using file handling operations.

------------------------------------------------------------------------

## 🚀 Features

-   ✍️ Add new journal entries\
-   📖 View all saved entries\
-   🔍 Search entries using keywords\
-   🗑️ Delete all entries\
-   ⚠️ Exception handling for error management

------------------------------------------------------------------------

## 🛠️ Technologies Used

-   Python\
-   OOP (Classes & Inheritance)\
-   File Handling (.txt file)\
-   Exception Handling

------------------------------------------------------------------------

## 🧠 OOP Concepts Used

-   **Class & Object**
-   **Inheritance**
-   **Encapsulation**

------------------------------------------------------------------------

## 📂 File Handling Modes

-   `r` → Read file\
-   `w` → Write (overwrite file)\
-   `a` → Append new data

------------------------------------------------------------------------

## ▶️ How to Run the Project

1.  Open Jupyter Notebook\
2.  Copy the code into cells\
3.  Run all cells\
4.  Use the menu to perform operations

------------------------------------------------------------------------

## ⚠️ Error Handling

-   Handles file not found errors\
-   Prevents crashes using try-except blocks
